# Reporting Standards

## Relevance

Structured reporting artifacts help reviewers separate intended use, evidence, limitations, and unsupported claims.

## Card Fields Strengthened

- `intended_use`
- `out_of_scope_uses`
- `evidence_bundle`

## Starter Resources

- [Model Cards for Model Reporting](https://arxiv.org/abs/1810.03993): establishes a model-reporting precedent for intended use, limitations, and evaluation evidence.

## Expansion Targets

- Add system cards, dataset documentation, evaluation cards, and audit packets.
