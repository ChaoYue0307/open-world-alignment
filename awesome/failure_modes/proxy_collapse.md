# Proxy Collapse

## Relevance

Proxy collapse happens when a proxy measure is treated as the real target-world objective.

## Card Fields Strengthened

- `target_world_state`
- `claim_boundaries`
- `hard_constraints`

## Starter Resources

- [Goodhart's law](https://en.wikipedia.org/wiki/Goodhart%27s_law): summarizes why optimized measures can stop tracking the intended target.
