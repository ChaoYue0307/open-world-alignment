# Claim Type and Measurement Validity

- [HELM: Holistic Evaluation of Language Models](https://arxiv.org/abs/2211.09110): supports multi-scenario, multi-metric claims instead of single-score generalization.
- [Evaluating Generative AI Systems Is a Social Science Measurement Challenge](https://arxiv.org/abs/2502.00561): frames evaluation as construct measurement, useful for `claim_type` and `target_world_state`.
- [CheckList](https://aclanthology.org/2020.acl-main.442/): demonstrates behavioral tests that make capability claims more specific.
- [Dynabench](https://aclanthology.org/2021.naacl-main.324/): motivates dynamic evaluation when static benchmark claims decay.
