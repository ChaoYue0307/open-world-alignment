# Simulation-Based Evaluation

## Relevance

Simulators can stress-test systems before deployment, but simulator assumptions must be reported and monitored.

## Card Fields Strengthened

- `external_tools_or_simulators`
- `uncertainty`
- `monitoring_and_rollback`

## Starter Resources

- [CARLA](https://carla.org/): provides a simulator example for embodied and autonomous-driving evaluation.
