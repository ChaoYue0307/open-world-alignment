# Benchmark Validity

- [Measuring Massive Multitask Language Understanding](https://arxiv.org/abs/2009.03300): useful for distinguishing broad coverage from deployment competence.
- [Measuring Coding Challenge Competence With APPS](https://arxiv.org/abs/2105.09938): illustrates coding benchmark construction and limits.
- [BIG-bench](https://arxiv.org/abs/2206.04615): provides a broad task collection while motivating caution about heterogeneous aggregate scores.
- [Benchmark Data Contamination of Large Language Models](https://arxiv.org/abs/2310.17589): supports card fields on unsupported claims and contamination risks.
