# Evaluation Debt

## Relevance

Evaluation debt appears when systems, deployments, or claims evolve faster than the evidence behind them.

## Card Fields Strengthened

- `system_version`
- `evidence_bundle`
- `date_updated`

- [The ML Test Score](https://research.google/pubs/the-ml-test-score-a-rubric-for-ml-production-readiness-and-technical-debt-reduction/): provides a rubric for production-readiness and technical-debt reduction in ML systems.
