# Multi-Agent Systems

## Relevance

Multi-agent behavior can produce emergent incentives and failures that are invisible when each agent is evaluated alone.

## Card Fields Strengthened

- `target_world_state`
- `absent_stakeholders`
- `uncertainty`

## Starter Resources

- [Melting Pot 2.0](https://arxiv.org/abs/2211.13746): provides multi-agent environments for evaluating social and strategic behavior.
