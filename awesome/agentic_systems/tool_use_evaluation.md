# Tool-Use Evaluation

## Relevance

Tool-using agents require checks for correct tool selection, permission boundaries, side effects, and reproducible execution.

## Card Fields Strengthened

- `external_tools_or_simulators`
- `hard_constraints`
- `evidence_bundle`

## Starter Resources

- [ReAct](https://arxiv.org/abs/2210.03629): links reasoning and acting traces, making tool-use behavior more auditable.

## Expansion Targets

- Add browser, shell, API, database, simulator, and retrieval evaluation resources.
