# Agent Evaluations

- [SWE-bench](https://arxiv.org/abs/2310.06770): anchors coding-agent claims in repository-level tasks and executable checks.
- [WebArena](https://arxiv.org/abs/2307.13854): evaluates autonomous web agents in realistic web environments.
- [AgentBench](https://arxiv.org/abs/2308.03688): provides a multi-environment benchmark suite for language agents.
- [OSWorld](https://arxiv.org/abs/2404.07972): evaluates agents on computer-use tasks where action logs and rollback matter.
