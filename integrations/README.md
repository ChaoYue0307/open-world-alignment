# Integrations

These notes map Open-World Evaluation Cards to existing evaluation frameworks. They are lightweight integrations: no framework is reimplemented here, and no framework is treated as sufficient evidence for open-world alignment by itself.

## Available Notes

- [Inspect AI](inspect_ai/README.md)

- [NIST AI RMF mapping](nist_ai_rmf/owa_to_nist_ai_rmf.md)

## Mapping Rule

An integration should answer three questions:

1. Which card fields can this framework provide evidence for?
2. Which claims remain unsupported?
3. What monitoring, rollback, or stakeholder evidence is outside the framework?
