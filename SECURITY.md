# Security Policy

This repository is an evaluation artifact toolkit. It does not host a deployed AI system, model endpoint, production service, or security-critical runtime.

## What to Report

Please open an issue if you find:

- a validator bug that accepts structurally invalid cards;
- a schema inconsistency that could mislead downstream users;
- documentation that could encourage unsafe evaluation or deployment practice;
- a resource link that points to malicious or misleading content.

## What Is Out of Scope

- Security vulnerabilities in third-party papers, benchmarks, models, or tools linked from the repository.
- Claims that a filled example card is insufficient for a real deployment without additional domain-specific controls.
- Requests for private disclosure about deployed systems not maintained in this repository.

For urgent issues, contact the repository owner through the GitHub profile associated with this repository.
