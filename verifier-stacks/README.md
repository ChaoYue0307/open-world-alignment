# Verifier Stacks

Verifier stacks describe the evidence channels that support or limit an Open-World Evaluation Card. They are not complete benchmark implementations. They are checklists for deciding whether a claim has enough independent, world-facing evidence.

Each domain file follows the same headings:

- Domain Claim
- Human Signals
- Human-Signal Limits
- Hard Verifiers
- Soft Verifiers
- Independence Rationale
- Verifier-Gaming Risks
- Hard Constraints
- Escalation Conditions
- Monitoring Signals
- Rollback Triggers
