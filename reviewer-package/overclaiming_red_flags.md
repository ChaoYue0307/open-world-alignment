# Overclaiming Red Flags

- A leaderboard score is used as evidence for deployment readiness.
- Human preference approval is used as evidence of truth, safety, or long-horizon benefit.
- The card does not list unsupported claims.
- The verifier stack contains only one type of evidence.
- Soft model judges are treated as independent hard verifiers.
- Constraints are prose-only and cannot block a claim.
- Monitoring is listed but no authority can revise or withdraw the claim.
- Affected non-users are absent from the evaluation boundary.

When a red flag appears, reviewers should ask for narrower claims, additional verifiers, or explicit unsupported-claim language.
