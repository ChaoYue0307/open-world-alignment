# Reviewer Package

This package is for reviewers who need to assess an Open-World Evaluation Card, benchmark claim, or paper appendix quickly.

## Use by Time Budget

- [5-minute checklist](reviewer_checklist_5min.md): decide whether the claim is obviously overextended.
- [15-minute checklist](reviewer_checklist_15min.md): inspect the eight required fields and evidence bundle.
- [Full checklist](reviewer_checklist_full.md): artifact-level review for cards, benchmarks, and deployment reports.

## Red Flags

- [Overclaiming red flags](overclaiming_red_flags.md)
- [Desk-reject red flags](desk_reject_red_flags.md)

For scoring, use the [reviewer rubric](../rubrics/reviewer_rubric.md). For definitions, use the [specification](../SPEC.md).
