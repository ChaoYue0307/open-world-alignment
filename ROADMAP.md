# Roadmap

## v0.1: Core Card Toolkit

- Open-World Evaluation Card template.
- JSON-compatible YAML template.
- JSON Schema.
- Validator and unit tests.
- Initial filled examples.
- Reviewer checklist.

## v0.2: Mini-Standard

- `SPEC.md`.
- Validation levels.
- Schema v0.2 optional audit metadata.
- Card variants for systems, benchmarks, deployments, agents, and model releases.
- Field guidance and maturity rubric.
- Quality linting, scoring, rendering, and link/resource checks.

## v0.3: Domain and Integration Library

- Expanded domain examples.
- Verifier-stack taxonomy.
- Failure-mode library.
- Reviewer package.
- Lightweight mappings for Inspect AI, OpenAI Evals, SWE-bench, and NIST AI RMF.

## v1.0: Stable Research Artifact

- Frozen required field set.
- Stable schema compatibility policy.
- Documentation site.
- Audit-ready examples.
- Optional release and DOI workflow, if requested.
